# CollabCode

A real-time collaborative code editor that allows multiple users to write and edit code simultaneously. Built with React, Socket.IO, and CodeMirror, featuring live synchronization across all connected clients.

## Features

- **Real-time Collaboration**: Multiple users can edit code simultaneously with instant synchronization
- **Multi-language Support**: Syntax highlighting for JavaScript, Python, Java, HTML, CSS, and more
- **File Management**: Create, upload, and switch between multiple files
- **Live Updates**: See changes from other users in real-time
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Dark Theme**: Easy on the eyes with CodeMirror's One Dark theme

### Frontend

- **React TypeScript** - Language
- **Vite** - Fast build tool
- **CodeMirror** - Code editor component
- **Socket.IO Client** - Real-time communication
- **React Router** - Navigation
- **Bootstrap** - Styling

### Backend

- **Node.js** - Runtime environment
- **Express** - Web server framework
- **Socket.IO** - WebSocket server
- **TypeScript** - Language
- **Multer** - File upload handling

## Tech used:

- [CodeMirror](https://codemirror.net/) - The code editor component
- [Socket.IO](https://socket.io/) - Real-time communication
- [Render](https://render.com/) - Backend hosting
- [Vercel](https://vercel.com/) - Frontend hosting
